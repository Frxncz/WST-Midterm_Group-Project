HTML

Hypertext Markup Language

1. Intro to HTML

Get introduced to HTML, the core language used to structure web content.

2. Elements & Tags

Learn about HTML elements and tags, which create and organize content on web pages.

3. HTML boilerplate

Set up a standard HTML structure with essential tags like < !DOCTYPE html >, < html >, and < body >.

4. Headings & List

Use headings and lists to create organized, readable web content.

5. Links

Learn to create links with the < a > tag, enabling users to navigate between pages.

6. Images

Embed images using the tag, adding visual elements to web pages.

CSS

Cascading Style Sheets

1. Intro to CSS

Learn the basics of CSS, a styling language that enhances the look of HTML by adding colors, fonts, and layouts.

2. The Cascade

Understand how CSS determines which styles apply, focusing on specificity, importance, and inheritance.

3. CSS Selectors

Explore different CSS selectors to target and style specific HTML elements effectively.

4. The Box Model

Learn about the box model, which controls the content, padding, border, and margin for layout and spacing.

5. Colors

Discover how to add colors to web pages using named colors, hex codes, RGB, and HSL formats.

6. Block & Inline

Understand the difference between block and inline elements and how they affect layout.
